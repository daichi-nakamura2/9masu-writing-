# 9マス式ライティングゲーム オンライン版

「何を書くか」ではなく **「どの質問に答えるか」**。
AIは文章を書きません。質問（9マス）を出すだけで、答えを埋めていくと文章が完成する教育ゲームのオンライン版です。
文章術だけでなく、インタビュー力・自己分析・企画書づくり・プレゼン準備の練習にもなります。

## 遊び方（ワークショップの流れ）

1. **ファシリテーター**が「部屋を作成」してテーマを決める
2. **プレイヤー**が部屋コードで参加する（※参加はロビー中のみ）
3. ファシリテーターが**テーマ**と**質問セット**を選んで「記入タイム」を開始
   （テーマを選ぶと、そのテーマに適した質問セットへ自動で切り替わります）
4. 各自が **9マスの質問**に答える（感情／エピソード／読者目線カード）
5. **テンプレート**を選ぶと、自分の答えを並べた**下書き**が自動生成される
   （並べているのは本人の言葉だけ。AIは書きません）
6. 下書きを整えて **文章を完成**させる
7. ファシリテーター画面で全員の進み具合を見ながら「終了」を押す
8. 一人ずつ**発表** → ほかの全員が **5項目×2点＝10点**で採点
9. **ランキング発表**（基本点＋投票点で称号が決まる）

## スコア

- **基本点（最大60点）**：9マスすべて記入で60点。途中でも記入数に応じて加点。
- **投票ボーナス（最大40点）**：他プレイヤーの平均点（0〜10点）を0〜40点に換算。
- 合計 **100点満点**。

### 称号

| 点数 | 称号 |
|---|---|
| 95〜100 | 質問マスター |
| 80〜94 | 文章職人 |
| 60〜79 | ストーリービルダー |
| 40〜59 | 文章探検家 |
| 20〜39 | 質問コレクター |
| 0〜19 | 見習い質問家 |

## 画面

- **プレイヤー画面**：テーマ → 9マス → テンプレート → 完成文章。他プレイヤーの記入内容は見えません。
- **ファシリテーター画面**：全員の9マス完成率・テンプレート選択・完成状態を一覧表示。「終了」で全員一斉に発表へ。

## ローカルで動かす

```
npm install
npm start
```

ブラウザで http://localhost:3003 を開く（環境変数 `PORT` で変更可）。

## 遠隔で遊ぶ（デプロイ）

`render.yaml` を同梱しています。GitHub にアップして [Render](https://render.com)（無料）に
Blueprint として読み込ませると、固定URLで24時間アクセスできます。
手順は姉妹アプリ `../yuuki-no-shizuku/DEPLOY.md` と同じ流れです（名前を `9masu-writing` に読み替え）。

## カスタマイズ

`server.js` 冒頭の定数を書き換えるだけで拡張できます。

- `QUESTION_SETS` … テーマの種類別の質問セット（体験／人・感謝／好き・こだわり／場所／変化の5種）。
  各セットは9問。テンプレートと噛み合うよう **index7＝数字／index8＝伝えたいこと** に統一してある
- `THEMES` … テーマ一覧（`set` = そのテーマにおすすめの質問セット。プリセット選択時に自動で切り替わる）
- `TEMPLATES` … 文章テンプレート（`order` は9マスの並べ替え順。フロントにも自動で反映されます）
- `VOTE_CATEGORIES` … 採点項目（各0〜2点）
